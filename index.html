About this demo: This flocking simulation is written in C, compiled to WebAssembly using Emscripten, and rendered with WebGL. The spatial grid partitioning reduces complexity from O(n²) to O(n), enabling smooth performance with hundreds of particles. Original JavaScript version at ~30 FPS, WebAssembly version targets 60 FPS.
